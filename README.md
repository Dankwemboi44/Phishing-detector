# Phishing Detector for Kiswahili & Sheng Code-Switched Text

## Overview
A machine learning system that detects phishing messages written in Kiswahili, Sheng, and code-switched text common in Kenya.

## Features
- Preprocessing tailored for Sheng slang
- TF-IDF + Logistic Regression baseline
- LSTM deep learning model
- Simple web interface for testing
